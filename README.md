# 🗣️ AI Speech Coach App

An AI-powered web app that helps users improve their speaking skills by analyzing their recorded or uploaded speech. It provides transcription, AI-generated summaries, and personalized feedback using modern web technologies.

## 🚀 Features

- 🎙️ Record or upload speech audio
- 📝 Transcribe speech using OpenAI Whisper
- 🤖 Summarize speech with Gemini (tone, clarity, and delivery feedback)
- 📊 Visualize performance metrics (e.g., pace, filler words)
- 📁 Review past sessions with insights and analytics

## 🛠 Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Django, Django REST Framework
- **Database:** PostgreSQL
- **AI Services:** OpenAI Whisper & Gemini APIs
- **Charts:** Recharts
